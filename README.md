# 🔐 Random Password Generator

A simple web-based Random Password Generator built using Python, Flask, HTML, and CSS. It generates secure passwords based on the user-defined length.

## Features

- Generate strong random passwords
- Custom password length
- Includes letters, numbers, and special characters
- Simple and responsive UI

## Technologies Used

- Python
- Flask
- HTML
- CSS

## Run Project

```bash
pip install flask
python app.py
